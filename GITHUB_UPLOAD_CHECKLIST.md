# GitHub 最終上傳清單

以下檔案放在 repository 根目錄即可直接部署到 GitHub Pages。

## 必要檔案

- `index.html`
- `styles.css`
- `404.html`
- `sitemap.xml`
- `chapter-01.html` 到 `chapter-42.html`

## 建議一起上傳

- `README.md`
- `GITHUB_REPO_COPY.md`
- `.gitignore`

## 不可漏掉的重點

- 所有檔案都放在 repository 根目錄
- `index.html` 必須存在，這是首頁
- `styles.css` 必須存在，否則頁面樣式會消失
- `404.html` 可讓 GitHub Pages 顯示正式找不到頁面

## 發布前檢查清單

- `index.html` 能正常開啟
- 42 個章節頁都存在
- 首頁卡片可以點進各章
- 每章的上一章 / 下一章按鈕可正常跳轉
- `404.html` 存在
- `sitemap.xml` 已改成正式網站網址

## 建議上傳順序

1. 上傳 `index.html`
2. 上傳 `styles.css`
3. 上傳 `404.html`
4. 上傳 `sitemap.xml`
5. 上傳 `chapter-01.html` 到 `chapter-42.html`
6. 最後補上 `README.md` 與 `GITHUB_REPO_COPY.md`
