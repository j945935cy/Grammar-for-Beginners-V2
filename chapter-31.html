Chapter 31｜some / any

這一章要學 some 和 any，讓你在肯定句、否定句和問句裡更自然地表達「一些」。

回首頁 上一章 下一章
學

本章學習目標

理解差別

知道 some 常見於肯定句，any 常見於否定句和問句。

會寫句型

能把 some / any 放進基本句子中。

避免混淆

減少問句和否定句用錯的情況。

1

白話解釋

some 和 any 都可以表示「一些」，但用的地方不太一樣。

基本規則

some：常用在肯定句

any：常用在否定句和問句

例如：I have some cookies. / I don't have any cookies. / Do you have any cookies?

2

常見錯誤

錯誤 1：否定句用 some

I don't have some money.

I don't have any money.

否定句常用 any。

錯誤 2：問句用 some

Do you need some help?

Do you need any help?

一般問句常用 any。

錯誤 3：肯定句用 any

She has any friends here.

She has some friends here.

肯定句常用 some。

3

正確句型

肯定句：主詞 + 動詞 + some + 名詞

I bought some snacks.

否定句：主詞 + don't / doesn't + 動詞 + any + 名詞

We don't have any eggs.

問句：Do / Does + 主詞 + 動詞 + any + 名詞 ... ?

Do you have any questions?

4

例句

  • I need some water. 我需要一些水。
  • There are some apples on the table. 桌上有一些蘋果。
  • We don't have any homework today. 我們今天沒有任何功課。
  • Does he have any brothers? 他有兄弟嗎？
  • She bought some flowers for Mom. 她買了一些花給媽媽。
5

練習題

練習說明：下面 5 題是改錯題，請把 some / any 改成正確形式。

  1. I don't have some books.
  2. Do you need any help?
  3. She has any pencils.
  4. We have some milk.
  5. Does Tom have some sisters?
答

答案與解析

  1. I don't have any books.

    解析：否定句用 any。

  2. Do you need any help?

    解析：這題本來就是正確句子。

  3. She has some pencils.

    解析：肯定句常用 some。

  4. We have some milk.

    解析：這題本來就是正確句子。

  5. Does Tom have any sisters?

    解析：一般問句常用 any。
