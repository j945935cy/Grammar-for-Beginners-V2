整

本週重點整理

介系詞與連接

時間、地點與句子延伸都是讓意思更完整的工具。

比較與完成

比較級在比較差異，完成式在看現在結果或經驗。

被動與修飾

被動看結果，關係子句看名詞後面的補充資訊。

條件句

if 子句常用現在式，結果句常用 will。

練

混合練習

  1. 改正：We stayed home because it rained, so we were bored.
  2. 改正：She has went to school already.
  3. 改正：The song sounds beautifully.
  4. 改正：If it will rain, we will cancel the game.
  5. 完成句子：This test is ___ than the last one.（easier / easiest）
  6. 完成句子：The bike ___ repaired yesterday.（was / is）
答

答案與解析

  1. We stayed home because it rained. / It rained, so we stayed home.

    解析：because 和 so 通常擇一使用。

  2. She has gone to school already.

    解析：go 的過去分詞是 gone。

  3. The song sounds beautiful.

    解析：sound 是連綴動詞，後面用形容詞 beautiful。

  4. If it rains, we will cancel the game.

    解析：if 子句常用現在式 rains。

  5. easier.

    解析：兩者比較，用比較級 easier。

  6. was.

    解析：yesterday 是過去時間，所以用 was repaired。
