## Repository 首頁簡介文案

這是一套以國中程度為主的英文文法網站教材，共 42 章，內容涵蓋基礎句型、時態、助動詞、詞類、問句、比較級到總複習與模擬練習。每一章都包含白話解釋、常見錯誤、正確句型、生活例句、練習題與答案解析，適合學生自學、老師課堂教學，以及作為 GitHub Pages 的靜態教材網站。

## 精簡版介紹

- 適合自學、複習與課堂使用
